# MAMDC2

Welcome to the **MAMDC2** repository. This repository contains two distinct spatial analysis pipelines:

* 🔬 **[Extracellular Matrix Analysis](./Extracellular%20matrix)**: A high-throughput pipeline for extracting ECM topological geometry and spatial interactions.
* 🥩 **[Muscle Fibres Analysis](./Muscle%20Fibres)**: Myotubes and nuclei analysis in images from cell cultures.

*Click on either folder above to view the specific code, requirements, and installation instructions for that project.*
